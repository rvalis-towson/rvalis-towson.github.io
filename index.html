Basics

Activities

  • Basically a "screen"
  • Most of your code can go here (though not suggested)
  • Loads and display a UI
  • Hook up event handlers here

Intents

  • Mechanism to start other apps and send/receive data

Resources

  • located in the /res directory
  • Layouts, images, other static data
  • Bundled with your app

UI

  • Declared in XML or the designer
  • LinearLayout - displays children in horizontal or vertical order
  • TextView - Displays text
  • RecyclerView - Displays a list of views

Folder Structure

Android relies on a very strictly defined folder structure. 

- /app
  - /src
    - /test -> non-ui tests
    - /androidTest -> ui tests
    - /main -> app code
      - /java -> java code
      - /res -> resources
      - AndroidManifest.xml -> Describes your app to the system
  - build.gradle -> your app's build definition

Resources (/res)

/res
   /layout -> your layout files
      activity_main.xml
   /values -> static values
      /strings.xml (static strings for text to be displayed)
      /colors.xml (colors used throughout your app)
      ... others
   /drawable -> images and icons

The App

HNews

  • Hacker news article fetcher
  • Displays the top 10 stories in a list
  • User can click a story to open the link in a browser
  • That's it!

For the brave

  • Try visually marking an item as "viewed"
  • Change each row's UI to be prettier
  • For the super brave, add a search bar to search the news

Components

  • One activity / screen. No rotation
  • RecyclerView to display the news
  • Intent to launch the browser link
  • OkHttpOkHttp to make the REST API request to fetch the data

MainActivity


  override fun onCreate(savedInstanceState: Bundle?) {
      super.onCreate(savedInstanceState)
      setContentView(R.layout.activity_main)

      val adapter = NewsAdapter(this)
      recyclerView.adapter = adapter
      recyclerView.layoutManager = LinearLayoutManager(this)

      fetchNews()
  }

Coroutines


    fun fetchNews() {
      scope.launch(Dispatchers.IO) {
          val items = HackerNewsApi().fetchNews()
          newsList.clear()
          newsList.addAll(items.value)
          withContext(Dispatchers.Main) {
              recyclerView.adapter?.notifyDataSetChanged()
              showListAndHideProgress()
          }
      }
